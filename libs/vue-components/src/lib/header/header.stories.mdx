## Overview

A header to use for a Government of Alberta microsite.  Contains a customizable service name and home link for the microsite.

## Examples
```
<template>
  <div id="app">
    <goa-header serviceName="Sample usage example" serviceHome="/" serviceLevel="Alpha" />
  </div>
</template>

<script>
import { GoAHeader } from '@abgov/vue-components';

export default {
  name: "App",
  components: {
    'goa-header': GoAHeader
  }
};
</script>
```
