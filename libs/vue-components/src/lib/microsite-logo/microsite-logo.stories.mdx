## Overview

A logo to use for a Government of Alberta microsite.  Contains a customizable service name and home link for the microsite.

## Examples
```
<template>
  <div id="app">
    <goa-microsite-logo serviceName="Sample usage example" serviceHome="/" />
  </div>
</template>

<script>
import { GoAMicrositeLogo } from '@abgov/vue-components';

export default {
  name: "App",
  components: {
    'goa-microsite-logo': GoAMicrositeLogo
  }
};
</script>
```
