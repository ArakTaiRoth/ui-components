# Radio Button

### Radio buttons are contained in a fieldset for accessibility and categorization purposes. The option group (radio button and label) can be selected by the user with an input device. The option is meant to be submitted as data and is part of a form. The input data, while can be selected, will not be recorded until the user presses a submission button.

[Figma design component](https://www.figma.com/file/3pb2IK8s2QUqWieH79KdN7/DIO-DS---Components?node-id=102%3A26)

export const BasicTemplate = () => {
  function onChange(name, value) {
    console.log('onChange', name, value);
  }
  return (
    <GoARadioGroup name="basic" value="orange" onChange={onChange}>
      <GoARadioItem name="basic" label="Red" value="red" />
      <GoARadioItem name="basic" label="Blue" value="blue" />
      <GoARadioItem name="basic" label="Orange" value="orange"/>
    </GoARadioGroup>
  )
}

<Story name="Basic">{BasicTemplate.bind({})}</Story>

**Note:** The `name` prop is required and must be the same for the `GoARadioGroup` and `GoARadioItem` components.
