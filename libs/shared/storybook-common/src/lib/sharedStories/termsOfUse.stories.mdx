# Terms of Use

Terms of Use for GoA visual assets and code for public-facing application projects.

The GoA Design System is for use on active development projects for public-facing applications, including:

- Open applications
- Secure applications
- Extranets/SharePoint sites for general public or ministry stakeholder audiences

The GoA Design System should not be used on:

- Program websites currently in old templates
- Intranets
- Internal SharePoint sites
