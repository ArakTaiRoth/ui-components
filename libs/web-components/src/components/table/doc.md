# Table Library
Table is a set of structured data that is easy for a user to scan, examine, and compare.

Use it like this:
```html
<goa-table>
  <thead>
    <tr>
      <th>Col 1</th>
      <th>Col 2</th>
      <th>Col 3</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Item 1</td>
      <td>Item 2</td>
      <td>Item 3</td>
    </tr>
    <tr>
      <td>Item 4</td>
      <td>Item 5</td>
      <td>Item 6</td>
    </tr>
  </tbody>
</goa-table>
```
